# Model card: `synthetic-acoustic-centroids-v1`

## Summary

This is a deterministic nearest-centroid teaching model for three neutral
synthetic waveform families:

- `steady_tone`: a continuous generated sine wave;
- `pulsed_tone`: a generated sine wave with a regular on/off envelope; and
- `broadband_texture`: deterministic seeded pseudorandom noise.

It is **not diagnostic**, was not trained on human recordings, and cannot
evaluate health, identity, intent, authenticity, quality, or safety.

## Training data

The model constructs six procedural variants per class at initialization. Each
variant is generated by `acoustic_sandbox.synthetic` at 16 kHz. No downloaded
dataset, third-party audio, microphone recording, or user upload contributes to
the centroids.

## Features and algorithm

The standard-library WAV pipeline computes RMS energy, peak amplitude, crest
factor, zero-crossing rate, mean absolute sample delta, and active-sample ratio.
Feature vectors are standardized with global synthetic-data scales. The model
selects the nearest class centroid and exposes normalized inverse-distance
weights as `relative_scores`.

Those scores are similarities within this toy model. They are not calibrated
probabilities or confidence about real-world sound sources.

## Intended uses

- demonstrate a browser-to-WSGI audio path;
- test upload validation and ephemeral processing;
- teach deterministic feature extraction and centroid classification; and
- provide safe integration fixtures without recorded audio.

## Out-of-scope uses

- diagnosis, screening, treatment, triage, or wellbeing assessment;
- classifying coughs, speech, people, diseases, or medical conditions;
- biometric identification, emotion recognition, surveillance, or moderation;
- safety-critical automation; and
- presenting regression-test behavior as real-world model accuracy.

## Evaluation and limitations

Tests verify that held-out variants from the same three generators map back to
their generator family. That is a software regression check, not evidence of
generalization. Real recordings may map arbitrarily because microphone
response, noise, clipping, codecs, mixed sources, and signal structure were not
represented during synthetic centroid construction.

## Versioning

Any change to feature definitions, synthetic generators, normalization, class
set, or distance calculation must use a new model identifier.
